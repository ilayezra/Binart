Convert Image to Binary

⚠️

Important things to note before using image to binary convertion:

  • Only .png image file format is allowed
  • Each pixel in your image must be 1x1 and not scaled up
  • The image must consist only 2 colors, otherwise it will pick 2 random colors
  • An image cannot consist more than 50,000 pixels.
